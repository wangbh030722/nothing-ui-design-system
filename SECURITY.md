# Security Policy

## Supported Version

Security fixes are applied to the latest version on the `main` branch.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Use GitHub's private vulnerability reporting feature on this repository. Include
the affected file, reproduction steps, impact, and any suggested mitigation.

You can expect an acknowledgement within seven days. The report will remain
private until a fix or coordinated disclosure plan is ready.
