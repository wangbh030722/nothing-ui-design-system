# Code of Conduct

## Our Standard

We are committed to a respectful, welcoming, and harassment-free community.
Examples of positive behavior include constructive feedback, empathy, clear
technical discussion, and respect for different experience levels.

Unacceptable behavior includes harassment, discrimination, personal attacks,
deliberate intimidation, publishing private information, or persistent
disruption of project discussions.

## Enforcement

Project maintainers may edit, remove, or reject contributions and communication
that violate this standard. Serious or repeated violations may result in a
temporary or permanent ban from project spaces.

Report conduct concerns privately through the security contact described in
`SECURITY.md`. Reports will be reviewed promptly and handled with discretion.

## Scope

This code of conduct applies in project repositories, issue trackers, pull
requests, discussions, and public spaces where someone represents the project.
